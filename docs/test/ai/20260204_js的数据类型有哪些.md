JavaScript 中的数据类型主要分为两大类：**原始类型 (Primitive Types)** 和 **对象类型 (Object Type)**。JavaScript 共有 8 种数据类型，其中 7 种是原始类型，1 种是对象类型。

**原始类型 (Primitive Types)** 包括：

1.  **String (字符串)**：用于表示文本数据，由单引号、双引号或反引号包裹的字符序列。
2.  **Number (数字)**：用于表示整数或浮点数。与 Java 等语言不同，JavaScript 不区分整数、浮点数等，统一使用 Number 类型。它还包括特殊的数值，如 `NaN` (Not a Number) 和 `Infinity`。
3.  **BigInt (大整数)**：用于表示和操作非常大的整数，这些整数超出了 Number 类型能够安全表示的范围。
4.  **Boolean (布尔值)**：表示逻辑实体，只有两个值：`true` (真) 和 `false` (假)。
5.  **Undefined (未定义)**：当变量被声明但未赋值时，其默认值就是 `undefined`。
6.  **Null (空)**：表示空值或“无”值。它是一个只有一个值 (`null`) 的特殊原始类型。JavaScript 中 `typeof null` 会返回 "object"，这是一个历史遗留的 bug。
7.  **Symbol (符号)**：ES6 中新增的类型，表示独一无二的值，常用于为对象属性创建唯一的键。

**对象类型 (Object Type)** 只有一种：

1.  **Object (对象)**：这是 JavaScript 中所有非原始值的基石。它是一个复杂的数据类型，用于存储键值对的集合。数组 (Array) 和函数 (Function) 在 JavaScript 中也都是特殊类型的对象。
